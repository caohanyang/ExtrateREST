The following table lists the releases for the Account API:

Version

Release Date

Summary of Release

Version 1.1.0

2017-07-28

A new call, getRateTables , was added to the API.

An expansion to Version 1.00 that includes new support for the AU and CA eBay marketplaces.

Version 1.1.0

This is a minor release of the API.

This API release contains the following updates and changes:

New calls

This release contains the following new call:

Version 1.0.0

The initial release of the Account API, Version 1.0.0, was released on October 19, 2016.

For a list of the new calls, see the Account API Reference .

Expanded functionality

Version 1.0.0 was updated on April 20, 2017 to include support for the following additional eBay marketplaces:

  • Australia (AU)
  • Canada (CA)
