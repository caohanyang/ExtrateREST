E-commerce Stores

Connect your E-commerce Store to MailChimp to take advantage of powerful reporting and personalization features and to learn more about your customers .

  • Carts
  • Customers
  • Orders
  • Products

    • POST /ecommerce/stores Add a new store

      Add a new store

      POST /ecommerce/stores

      Add a new store to your MailChimp account.

      Request body parameters

      id Type: String Title: Store Foreign ID Read only: false The unique identifier for the store.
      list_id Type: String Title: List ID Read only: false The unique identifier for the MailChimp List associated with the store. The list_id for a specific store cannot change.
      name Type: String Title: Store Name Read only: false The name of the store.
      platform Type: String Title: Platform Read only: false The e-commerce platform of the store.
      domain Type: String Title: Domain Read only: false The store domain.
      is_syncing Type: Boolean Title: Is Syncing Read only: false Whether the e-commerce store is currently syncing .
      email_address Type: String Title: Email Address Read only: false The email address for the store.
      currency_code Type: String Title: Currency Read only: false The three-letter ISO 4217 code for the currency that the store accepts.
      money_format Type: String Title: Money Format Read only: false The currency format for the store. For example: $ , £ , etc.
      primary_locale Type: String Title: Primary Locale Read only: false The primary locale for the store. For example: en , de , etc.
      timezone Type: String Title: Timezone Read only: false The timezone for the store.
      phone Type: String Title: Phone Read only: false The store phone number.
      address Type: Object Title: Address Read only: false The store address.
      Show properties
      address1 Type: String Title: Address Field 1 Read only: false The store’s mailing address.
      address2 Type: String Title: Address Field 2 Read only: false An additional field for the store’s mailing address.
      city Type: String Title: City Read only: false The city the store is located in.
      province Type: String Title: Province Read only: false The store’s state name or normalized province.
      province_code Type: String Title: Province Code Read only: false The two-letter code for the store’s province or state.
      postal_code Type: String Title: Postal Code Read only: false The store’s postal or zip code.
      country Type: String Title: Country Code Read only: false The store’s country.
      country_code Type: String Title: Country Code Read only: false The two-letter code for to the store’s country.
      longitude Type: Number Title: Longitude Read only: false The longitude of the store location.
      latitude Type: Number Title: Latitude Read only: false The latitude of the store location.
