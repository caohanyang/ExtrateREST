      Create a new list

      POST /lists

      Create a new list in your MailChimp account.

      Request body parameters

      name Type: String Title: List Name Read only: false The name of the list.
      contact Type: Object Title: List Contact Read only: false Contact information displayed in campaign footers to comply with international spam laws.
      Show properties
      company Type: String Title: Company Name Read only: false The company name for the list.
      address1 Type: String Title: Address Read only: false The street address for the list contact.
      address2 Type: String Title: Address Read only: false The street address for the list contact.
      city Type: String Title: City Read only: false The city for the list contact.
      state Type: String Title: State Read only: false The state for the list contact.
      zip Type: String Title: Postal Code Read only: false The postal or zip code for the list contact.
      country Type: String Title: Country Code Read only: false A two-character ISO3166 country code. Defaults to US if invalid.
      phone Type: String Title: Phone Number Read only: false The phone number for the list contact.
      permission_reminder Type: String Title: Permission Reminder Read only: false The permission reminder for the list.
      use_archive_bar Type: Boolean Title: Use Archive Bar Read only: false Whether campaigns for this list use the Archive Bar in archives by default.
      campaign_defaults Type: Object Title: Campaign Defaults Read only: false Default values for campaigns created for this list.
      Show properties
      from_name Type: String Title: Sender's Name Read only: false The default from name for campaigns sent to this list.
      from_email Type: String Title: Sender's Email Address Read only: false The default from email for campaigns sent to this list.
      subject Type: String Title: Subject Read only: false The default subject line for campaigns sent to this list.
      language Type: String Title: Language Read only: false The default language for this lists’s forms.
      notify_on_subscribe Type: String Title: Notify on Subscribe Read only: false The email address to send subscribe notifications to.
      notify_on_unsubscribe Type: String Title: Notify on Unsubscribe Read only: false The email address to send unsubscribe notifications to.
      email_type_option Type: Boolean Title: Email Type Option Read only: false Whether the list supports multiple formats for emails . When set to true , subscribers can choose whether they want to receive HTML or plain-text emails. When set to false , subscribers will receive HTML emails, with a plain-text alternative backup.
      visibility Type: String Title: Visibility Read only: false Whether this list is public or private .
      Possible Values:
      • pub
